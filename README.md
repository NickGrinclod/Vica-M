# Vica-M

![Vica-M](Vica-M.jpg)

## Overview

Vica-M is an autonomous mobile robot developed as part of the Robotics Engineering Technology program at George Brown College.

The platform is designed to perform autonomous navigation, obstacle avoidance, wall following, camera-based perception, and future object detection using ROS 2 and computer vision technologies.
